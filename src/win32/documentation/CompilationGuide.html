Siege of Avalon : Open Source Edition - Win32 Compilaton Guide


To be able to compile this project in Delphi 4 do the following...
  1. Go to menu "File/Open Projects..." and find and open the project "win32/source/Siege.dpr".
  2. Now go to menu "Project/Build Siege" and within a few seconds you should have the file "Siege.exe" created within the upper level "game" directory.
  3. Assuming all the data is set up correctly you should be able to start playing the game.
